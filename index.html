Trilhas Literárias de Hortolândia

Incentivo a leitura e acesso ao livro

Trilhas literárias de Hortolândia é um projeto que tem com objetivo atraves de cartazes com Qr code fornecer a população uma biblioteca atraves de seus smartphones e internet móvel para realizar download de livros de Domínio Público.

Neste site você encotra os sites de onde são tirados os livros e os locais onde poderá encontrar os cartazes.

O mapa

O mapa contem os pontos onde se encontra cartazes QR Code com links de livros

Esse site é um projeto que foi desenvolvido por Luiz Torres Junior para substituir o projeto Trilhas literárias de Hortolândia que era um projeto que tinha como objetivo criar uma maneira de tornar livros de dominio público acessivel a população da cidade de Hortolândia.
