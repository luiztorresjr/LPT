Trilhas Literárias de Hortolândia

Trilhas Literárias de Hortolândia é um projeto que tem como objetivo disponibilizar livros de domínio público
atraves de cartazes com Qr code fornecer a população uma biblioteca atraves de seus smartphones e internet
móvel para baixar livros.

Neste site você encontra os sites de onde são tirados os livros e os locais onde poderá encontrar os cartazes.

O mapa

O mapa contem os pontos onde se encontra cartazes QR Code com links de livros

Esse site é um projeto que foi desenvolvido por Luiz Torres Junior para substituir o projeto Trilhas literárias de Hortolândia que era um projeto que tinha como objetivo criar uma maneira de tornar livros de dominio público acessivel a população da cidade de Hortolândia.
