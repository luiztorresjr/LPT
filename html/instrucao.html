QR code como usar

O que é QR Code

QR Code é um tipo de código de barra, criado em 1999, no Japão, se tornou popular com o uso para marketing usando smartphones, pode ser gerado para direcionar smartphones para sites automaticamente.

Para sua leitura no smartphone é necessário possuir um aplicativo que possa ler, smartphones android hoje em dia veêm com leitura de QR Code embarcado na camêra do smartphone.

Para maiores informações sobre a história e como funciona o código QR Code Wikipedia Código QR

3º Coloque o QR Code no local indicado;
4º Aparecera o link aperte;
5º Aparecera o navegador e o site;
