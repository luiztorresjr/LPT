Sobre o projeto

A História do projeto

Esse projeto é derivado de um projeto criado durante o ano e 2015, chamado QR Code QR Code e dispositivos móveis para o acesso a literatura e informações utilitárias, sendo o ex-professor do IFSP campus Hortolândia Carlos Roberto dos Santos Junior responsável pelo auxilio no projeto, edital 006/2015 para maiores informações acesse o site nome do projeto e coordenador.

O projeto foi desenvolvido durante o ano de 2015 com o objetivos de criar uma forma de usar a tecnológia de QR Code para divulgar livros de dominío público para a população de Hortolândia, esse projeto foi desenvolvido com base no "projekt Pingeb" ou Projekt Ingeborg da cidade de Salzburgo na Áustria, onde usando no caso desse projeto etiquetas RFID(Radio-Frequency IDentification) para disbonililizar livro, para facilitar o acesso a livros para a população.
